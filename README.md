# 🎬 YouTube AI Assistant

A professional AI-powered tool that generates concise and structured summaries of YouTube videos using the Llama 3.3 model.

## ✨ Key Features
* **AI-Driven Summarization**: Automatically extracts core concepts and detailed insights from any video.
* **Responsive UI**: A modern web interface built with Streamlit, featuring smooth animations and a clean layout.
* **High-Performance Inference**: Utilizes the **Llama-3.3-70b** model via Groq API for near-instant results.
* **Bilingual Interface**: Supports both English and Russian UI toggles with multi-language summary output.
* **Smart Transcript Recovery**: Advanced handling of subtitles to ensure data is fetched even when standard methods are restricted.

## 🛠 Tech Stack
* **Python**: Core application logic.
* **Streamlit**: Dashboard and frontend elements.
* **Groq Cloud**: High-speed AI model hosting (Llama 3.3).
* **YouTube Transcript API**: Automated subtitle retrieval.
